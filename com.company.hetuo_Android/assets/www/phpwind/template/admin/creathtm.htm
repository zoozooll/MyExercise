提示信息

功能说明:
  1. 采用静态 html 方式来缓存主题内容从而大幅度节约数据库资源,并使浏览速度有质的飞跃.适合于大型网站
  2. 占用较多服务器空间,一般占用与您RDBMS同等空间
  3. 加密帖(出售,加密,隐藏)不会生成htm页面
  4. 如果您的空间打开 safe_mode 如果您对此不了解,建议不要使用此功能
  5. 使用此功能后用户的真实信息将滞后,请不定期重新生成所有 html 文件!
  6. 使用此功能后请不要采用多种风格.更不能使用图片.附件防盗链功能!
  7. 使用此功能后将自动启用点击率延迟更新功能

模板管理说明:
  1. 可选择一次性生成所有版块的静态模板，也可以选择生成单个版块的静态模板
  2. 当您取消某版块的静态模板功能时请删除相应的 html 数据,以节约空间!
