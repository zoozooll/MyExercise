Class Overview

A Pie Series displays magnitude data on the chart - the larger the value of the datapoint, the larger the slice representing that datapoint. Whereas data points for a CartesianSeries require x and y values, a pie series requires a name and value. The x value of a data point is used as the name of a slice, and the y value is used as its magnitude.

Styles for each individual slice can be configured in the PieSeriesStyle.
