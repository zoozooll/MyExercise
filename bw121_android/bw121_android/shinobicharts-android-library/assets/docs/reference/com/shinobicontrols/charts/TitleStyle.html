AxisTitleStyle, MainTitleStyle
AxisTitleStyle Controls the look and feel for the axis title. 
MainTitleStyle Visual style properties relating to the ShinobiChart title. 

Class Overview

Visual style properties for a title. If you make any styling changes after the chart has been displayed, you should call chart.redrawChart().
